# node-medium-api
Just another medium clone made with Node and Typescript


## Install application
```
$ npm install # yarn
```

## Build typescript
```
$ npm run build # yarn build
```

## Run server
```
$ npm run server # yarn server
```

### Migrations
```
$ typeorm migration:run
```

### Run seed file
```
$ npm run seed
```

#### More information
Know more about typeorm here: http://typeorm.io
